# Instructions to run the code(s):

- mentYOUR
- vidyabot

## Vidyabot Instructions

### Frontend
```
cd frontend
npm install --force
npm start
```


### API
```
cd api
. venv/bin/activate
pip install -r requirements.txt
python app.py
```

### CSV

Place the csv of questions with following columns :
```
question,answer_en,answer_hi,answer_mr
```
